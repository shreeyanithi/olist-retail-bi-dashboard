# Olist Retail Operations BI Dashboard
End-to-end business intelligence project on 100K+ Brazilian e-commerce transactions.

## Tools
Python · PostgreSQL · SQL · Tableau Public

## What's inside
- `notebook/` — Python EDA (pandas, matplotlib, seaborn)
- `sql/` — 5 analytical SQL scripts (CTEs, window functions, RFM, RANK)
- `outputs/` — cleaned CSVs and EDA charts

## Key Findings
- 8.1% late delivery rate nationally; AL state at 23% (3x average)
- Health & beauty is #1 revenue category
- Strong negative correlation between late delivery rate and seller review scores
